Table Checkable

Important To enable the checkable plugin, make sure you add the class .table-checkable to your table element.

Important Also, make sure you add the class .checkbox-column to each of your th, th elements that contain your checkboxes. This will likely be your very first column.


Example HTML

<table id="checkable-example" class="table table-bordered table-checkable">
	<thead>
		<tr>
			<th class="checkbox-column"><input type="checkbox"></th>
			<th>Column</th>
		</tr>
	</thead>

	<tbody>
		<tr>
			<td class="checkbox-column"><input type="checkbox"></td>
			<td>Column Text</td>
		</tr>
	</tbody>
</table>

Example JS

$('#checkable-example').tableCheckable ();

Calendar Plugins

FullCalendar is a jQuery plugin that provides a full-sized, drag & drop calendar. It uses AJAX to fetch events on-the-fly for each month and is easily configured to use your own feed format (an extension is provided for Google Calendar). It is visually customizable and exposes hooks for user-triggered events (like clicking or dragging an event). It is open source licensed under an MIT license.
