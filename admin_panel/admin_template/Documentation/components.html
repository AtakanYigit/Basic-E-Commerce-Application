Icons List

Example HTML

Important   Be sure that your ul has the class .icons-list.

Important   Be sure that the icon for each item has the class .icon-li in the class list. 

<ul class="icons-list">
	<li>
		<i class="icon-li fa fa-cloud"></i>
		Lorem ipsum dolor sit amet, consectetur adipiscing elit.
	</li>
	<li>
		<i class="icon-li fa fa-tasks"></i>
		Vestibulum iaculis felis eu eros pellentesque placerat.
	</li>
</ul>

Portlets

Portlet

Example HTML

Below is the example html for a portlet container. Portlets will usually go directly inside of a Bootstrap column, but could really go anywhere.


<div class="portlet">

	<div class="portlet-header">
		<h3>
			<i class="icon-asterisk"></i>
			Portlet Title
		</h3>
	</div>

	<div class="portlet-content">
		Lorem ipsum dolor sit amet, consectetur adipisicing elit.
	</div>

</div> <!-- /.portlet -->

Portlet Tools

Portlet tools can contain a variety of elements list below:

  • Regular Buttons
  • Dropdown Buttons
  • Split Buttons
  • Search Bar
  • Labels
  • Badges
  • Pill Tabs
  • Progress Bars
View Live Portlet Tools


Example HTML

<div class="portlet-header">
	<h3>
		<i class="icon-asterisk"></i>
		Portlet Title
	</h3>

	<ul class="portlet-tools pull-right">
		<li>
			<a href="javascript:;" class="btn btn-sm btn-primary">Action</a>
		</li>
	</ul>
</div> <!-- /.portlet-header -->

Pricing Tables

Layouts

The Pricing Tables component utilizes the Bootstrap grid for layouts.

<div class="row">

	<div class="col-md-4">

		<div class="pricing-plan"></div> <!-- .pricing-plan -->

	</div>

</div>

Example HTML

An example .pricing-plan element that you can use within a grid column.

<div class="pricing-plan">
	
	<h3 class="pricing-title">Basic</h3>
	
	<p class="pricing-label">For blogs & small business</p>   
	
	<hr>
	
	<span class="pricing-price">
		<span class="pricing-amount">$29</span> / month
	</span>
	
	<hr>
	
	<ul class="pricing-details">	    			
		<li><strong>10</strong> users</li>
		<li><strong>50</strong> projects</li>	   
		<li><strong>50BG</strong> of space</li>	   
		<li><strong>10</strong> email accounts</li>	  	 	    				
	</ul>
	
	<a href="javascript:;" class="btn btn-primary btn-lg">Choose plan</a>
	
</div> <!-- /.pricing -->

Details Popover

An example of the informational popup is shown below. This can be added to your ul.pricing-details element.

<li>
	<span class="pricing-help" 
			title="What's a user?" 
			data-content="Lorem ipsum dolor sit amet, consectetuer adipiscing elit.">
		<strong>100</strong> users
	</span>
</li>

Promoted Plan

To create a promoted plan add the class .promoted to your .pricing-plan element. You may also want to add the img tag below to the top of your .pricing element to denote the best value. 

<div class="pricing-plan promoted">

	<img src="./img/pricing/label-yellow.png" class="pricing-best-value" alt="Best Label" />

	<!-- Pricing Details Go Here -->

</div>

Thumbnails

Bootstrap Docs

Thumbnails Documentation

Image Hover

Image Hover is an extension of Bootstrap's thumbnail element. It easily allows you to have an icon to denote a zoom option when hovering over an image.

<div class="thumbnail-view">
	<a href="IMG_FULL_PATH" class="thumbnail-view-hover ui-lightbox"></a>
    <img src="IMG_THUMBNAIL_PATH" style="width: 100%" alt="Gallery Image">
</div>

Footer

The thumbnail footer provides an easy way to display comments, likes, and dates using icons.

<div class="thumbnail-footer">
	<div class="pull-left">
    	<a href="javascript:;"><i class="icon-thumbs-up"></i> 123</a> 
    	<a href="javascript:;"><i class="icon-thumbs-down"></i> 29</a>
    </div>

    <div class="pull-right">
    	<a href="javascript:;"><i class="icon-time"></i> 2 days ago</a>
    </div>
</div>
