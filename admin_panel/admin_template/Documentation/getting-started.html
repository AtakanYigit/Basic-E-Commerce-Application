Introduction

Canvas Admin is a responsive admin dashboard template powered with Twitter Bootstrap Framework for admin and backend applications. Canvas Admin has a clean and intuitive design which makes your next project look awesome and yet user friendly. Canvas Admin comes with a huge collection of plugins and UI components and it works in all major web browsers, tablets and phones.

View Theme Demo


Documentation Scope

This documentation is intended to explain basic template structure, custom elements, plugins, helper classes and/or how you can configure the template. Since this template is built on top of the Bootstrap framework, your main source of documentation can be found here: http://getbootstrap.com/css/.

This documentation is written in a style that assumes you already know the basics of Bootstrap, CSS, JavaScript and HTML. This document only covers additions, extensions, and elements pertinent to this theme.

Getting Started

To get started, just browse through the demonstration pages. Use Chrome Developer Tools (Chrome), Firebug (Firefox) or similar plugins to analyze the code. To get a deeper understanding of the code, open the HTML and JS files and read through the commented code.

Theme Structure

The below examples are not necessarily intended to be the correct order of the elements contained within this template. They are just examples to show you how to modify certain pieces.

Use the page-blank.html and start right away building your own pages.


Header

This is example html for the header section of the template. You can easily change the logo of the theme by modifying the h1#site-logo element.

	<h1 id="site-logo">
		<a href="./index.html">
			<img src="./img/logos/logo.png" alt="Site Logo" />
		</a>
	</h1>	

	<a href="javascript:;" 
			data-toggle="collapse" 
			data-target=".top-bar-collapse" 
			id="top-bar-toggle" 
			class="navbar-toggle collapsed">
		<i class="icon-cog"></i>
	</a>

	<a href="javascript:;" 
			data-toggle="collapse" 
			data-target=".sidebar-collapse" 
			id="sidebar-toggle" 
			class="navbar-toggle collapsed">
		<i class="icon-reorder"></i>
	</a>

</header> <!-- header -->

	    	</a>

	    	<ul class="dropdown-menu" role="menu">
		        <li><a href="javascript:;">Example #1</a></li>
		        <li><a href="javascript:;">Example #2</a></li>
		        <li class="divider"></li>
		        <li><a href="javascript:;">Example #3</a></li>
	    	</ul>
	    </li>
	    
	</ul>
</nav>

Content

This is the main content area of the theme.

Located within the #content-header element is an h1 tag which contains the main page title.

Important The #content-container element is where the bulk of content additions and/or changes will take place.

<div id="content">	

	<div id="content-header">
		<h1>Blank Page</h1>
	</div> <!-- #content-header -->	

	<div id="content-container">
		<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
	</div> <!-- /#content-container -->			
	
</div> <!-- #content -->

Javascript Helpers

Data Tables

Initialize

Important To utilize this helper, you must first include the Data Tables javascript files. Information on what files to include is located at the Data Tables plugin page

The Data Table helper allows you to use the plugin without dealing with javascript directly. To initialize the Data Table, add the attribute data-provide="datatable" to your table element.

<table data-provide="datatable">

Table Options

Name Attribute Values Description
Paginate data-paginate True/False Enables pagination for the table
Search data-paginate True/False Enables full table search
Table Info data-info True/False Displays information about visible rows and total rows for the table.
Row Length data-length-change True/False Allows you to change the number of visible rows in the table
Display Rows data-display-rows 10 Sets the default number of rows to display

Example HTML
<table data-provide="datatable"
		data-paginate="true"
		data-search="true"
		data-info="true"
		data-length-change="true"
		data-display-rows="10">

Column Options

Option Attribute Values Description
Filterable data-filterable True/False Enables a specific column to have basic text search
Sortable data-sortable True/False Enables basic sorting on a specific column
Direction data-direction Asc/Desc Sets the initial sorting value for column. Can be left blank.

Example HTML
<table data-provide="datatable">
	<thead>
		<tr>
			<th data-sortable="false" data-filterable="false" data-direction="desc">Column #1</th>
			<th data-sortable="true" data-filterable="true">Column #2</th>
			<th data-sortable="true" data-filterable="false">Column #3</th>
		</tr>
	</thead>
</table>

Charts

Most charting plugins require a defined height to display the chart correctly. With that being said the .chart-holder class sets a reasonable default height of 300px for your charts.

This plugin can be easily used with either jQuery Flot charts or Morris.js charts.

<div id="CHART_ID" class="chart-holder"></div> <!-- .chart-holder -->

Form Validation

To enable Parsley validation with custom error messages, add the class .parsley-form to your form.

Important This is different form the default Parsley method, which is using the attribute data-validate. You do not want to use the default Parsley method if utilizing the custom error messages.

<form class="form parsley-form">

</form>

Date Picker

Basic Input

To utilize the date picker helper, add the class .ui-datepicker to your input element.

<input type="text" name="datepicker" class="form-control ui-datepicker">

Picker Component

To utilize the date picker helper as a component, add the class .ui-datepicker to your .input-group element.

<div class="input-group date ui-datepicker">
    <input class="form-control" type="text">
    <span class="input-group-addon"><i class="icon-calendar"></i></span>
</div>

Time Picker

Picker Dropdown

To enable the Timepicker dropdown helper add the class .ui-timepicker to your input element.

<div class="input-group">
    <input type="text" class="form-control ui-timepicker">
    <span class="input-group-addon"><i class="icon-time"></i></span>
</div>

Modal Picker

To enable the Timepicker modal helper add the class .ui-timepicker-modal to your input element.

<div class="input-group">
    <input type="text" class="form-control ui-timepicker-modal">
    <span class="input-group-addon"><i class="fa fa-clock-o"></i></span>
</div>

Select2

To utilize the basic Select2 helper, add the class .select2-input to your select element.

<select name="select" class="form-control select2-input">
	<option value="">Please Select</option>
	<option value="1">1</option>
	<option value="2">2</option>
</select>

iCheck

To utilize the iCheck helper, add the class .icheck-input to your input:checkbox and input:radio elements.

<input type="checkbox" name="check" class="icheck-input" value="">
<input type="radio" name="radio" class="icheck-input" value="">

Color Picker

Important In order to utilize the following color pickers, you must select your desired color values. Each color value must have correct hexadecimal color format.

Inline Picker

To utilize the inline color picker helper, add the class .ui-colorpicker to your select element.

<select class="form-control ui-colorpicker">
  <option value="#e5412d">Red</option>
</select>

Dropdown Picker

To utilize the dropdown color picker helper, add the class .ui-colorpicker and the data attribute data-picker="true" to your select element.

<select class="form-control ui-colorpicker" data-picker="true">
  <option value="#e5412d">Red</option>
</select>

Textarea Autosize

To utilize the autosize helper, add the class .ui-textarea-autosize to your textarea element.

<textarea class="form-control ui-textarea-autosize" rows="3"></textarea>

Animate

You may animate the growth of the textarea by also adding the class .autosize-animate to your textarea element.

<textarea class="form-control ui-textarea-autosize autosize-animate" rows="3"></textarea>

Lightbox

Single Image Lightbox

To utilize the lightbox helper, add the class .ui-lightbox to the a element for the full sized image.

<a href="FULL_IMAGE_PATH" class="ui-lightbox">Open Larger Image</a>

Gallery Image Lightbox

To utilize the lightbox gallery helper, add a div with the class .ui-lightbox-gallery as a wrapper for the desired full image links.

<div class="ui-lightbox-gallery">
	<a href="FULL_IMAGE_PATH">Open Larger Image</a>
	<a href="FULL_IMAGE_PATH">Open Larger Image</a>
	<a href="FULL_IMAGE_PATH">Open Larger Image</a>
</div>

iFrame Lightbox

To utilize the lightbox iframe helper, add the class .ui-lightbox-iframe to the a element for the external page you would like to display.

<a href="GOOGLE_MAP_URL" class="ui-lightbox-iframe">Open Google Mape</a>

Tooltips

Popover Helper

To utilize the popover helper, add the class .ui-popover to the element you would like to connect the popover to.

<a href="javascript:;" 
		class="ui-popover" 
		title="Tooltip content here." 
		data-content="Popover content here.">
		Popover
</a>

Tooltip Helper

To utilize the tooltip helper, add the class .ui-tooltip to the element you would like to connect the tooltip to.

<a href="javascript:;" 
		class="ui-tooltip" 
		title="Tooltip content here.">
		Tooltip
</a>
